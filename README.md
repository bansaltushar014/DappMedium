### About
DappMedium is a platform where anybody can write their content and the reader will be paying directly to the content writer, no middleman involved. 

### Tools & Technology 
* Solidity
* Truffle framework
* Node js
* React
* Ganache
* Metamask 

### Installation Steps


#### First
* Setup Ganache at 7545
* Setup metamask at localhost:7545

#### Second
* git clone
* cd DappMedium
* npm install 
* cd frontend
* npm install
* cd ..
* npx truffle compile
* npx truffle migrate
* npm start
